# Библиотека - Веб-приложение для управления книгами

Веб-приложение для управления библиотекой книг с функциями добавления, обновления, удаления и поиска.

## Функциональность

- ✅ **Добавление книг** - добавление новых книг с информацией (название, автор, жанр, год, описание)
- ✅ **Обновление книг** - редактирование существующих записей
- ✅ **Удаление книг** - удаление книг из библиотеки
- ✅ **Поиск книг** - поиск по названию, автору или жанру в реальном времени
- ✅ **Локальное хранение** - все данные сохраняются в браузере (localStorage)

## Технологии

- HTML5
- CSS3 (с градиентами и анимациями)
- Vanilla JavaScript (ES6+)
- LocalStorage для хранения данных

## Использование

1. Откройте файл `index.html` в современном веб-браузере
2. Используйте форму для добавления книг
3. Используйте поле поиска для поиска книг
4. Нажмите "Редактировать" на карточке книги для изменения
5. Нажмите "Удалить" для удаления книги
